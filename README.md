# Mock E-Com Cart

A full-stack demo for shopping cart flows.

## Setup

- Backend: `cd backend && npm install && npm run dev`
- Frontend: `cd frontend && npm install && npm start`

## Features

- Product listing, cart, checkout receipt
- Responsive design
- RESTful API (Express/MongoDB)
- Demo modal receipt
